Var Hoisting

Var is hoisted in memory by javascript. While calling it before initiallize it is always undefined.

console.log("a", a);
var a = 10;
a undefined

Function Hoisting

Function is hoisted in memory by javascript.
Note: Arrow function Or A function expression where function is assigned to a variable will note be hoisted
b();
function b () { console.log("hi") }
hi

Arrow Function/ Function Expression Hoisting

Note: Arrow function Or A function expression assigned to let or const will be hoisted in temporal dead zone hence throw reference error.
While assigning function to var will be hoisted but hoisted as undefined hence throw TypeError: b is not function b();
let b = () => { console.log("hi") }
Uncaught ReferenceError: b is not defined
